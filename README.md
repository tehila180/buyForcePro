# 🛒 BuyForce – פלטפורמת רכישה קבוצתית

## 📌 תיאור כללי
המערכת BuyForce היא מערכת Full-Stack לרכישה קבוצתית, המאפשרת למשתמשים להצטרף לקבוצות רכישה של מוצרים, לבצע תשלום מאובטח באמצעות PayPal, ולסגור קבוצה רק כאשר כל המשתתפים שילמו.

המערכת פרוסה ופועלת בענן (Railway), ולכן אין צורך להפעיל שרת מקומי לצורך שימוש רגיל.

---

## 🎯 מטרות הפרויקט
- מימוש מערכת רכישה קבוצתית מקצה לקצה  
- עבודה עם אימות משתמשים (JWT)  
- שילוב מערכת תשלומים מאובטחת (PayPal)  
- פיתוח Web ו-Mobile  
- עבודה עם Backend פרוס בענן  

---

## 🧩 פיצ'רים עיקריים

### 👤 משתמשים
- הרשמה והתחברות למערכת  
- אימות באמצעות JWT  
- זיהוי משתמש מחובר / לא מחובר  
- הצגת הודעה למשתמש לא מחובר בעת ניסיון הצטרפות לקבוצה  

### 📦 מוצרים וקטגוריות
- צפייה ברשימת מוצרים  
- סינון מוצרים לפי קטגוריות  
- עמוד מוצר ייעודי  

### 👥 קבוצות רכישה
- הצטרפות לקבוצות רכישה  
- הצגת רשימת משתתפים בקבוצה  
- מעקב אחר סטטוס תשלום לכל משתתף  
- קבוצה נסגרת רק לאחר שכל המשתתפים שילמו  

### 💳 תשלומים (PayPal)
- יצירת תשלום באמצעות PayPal  
- תשלום מאובטח (Sandbox)  
- אישור תשלום (Capture)  
- טיפול בביטול תשלום  
- חסימת הצטרפות ותשלום למשתמשים לא מחוברים  

### 🛠️ ניהול (Admin)
- ניהול קטגוריות  
- ניהול מוצרים  
- ניהול קבוצות רכישה  
- ניהול משתמשים  
- צפייה בסטטיסטיקות  

---

## 🏗️ מבנה הפרויקט

### Backend (שרת)
- Node.js  
- NestJS  
- PostgreSQL  
- Prisma ORM  
- JWT Authentication  
- PayPal REST API  
- פריסה בענן באמצעות Railway  

ℹ️ השרת כבר רץ בענן ואין צורך להפעיל אותו מקומית (הרצה מקומית נדרשת רק לצורכי פיתוח).

---

### Frontend (Web)
- React  
- תקשורת ישירה מול ה-Backend הפרוס  
- טיפול בשגיאות הרשאה (401)

---

### Mobile
- React Native (Expo)  
- שימוש ב-WebView לתשלומי PayPal  
- תמיכה ב-Android ו-iOS
